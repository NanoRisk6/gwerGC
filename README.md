# GWERGC

GWERGC is a fast, browser-based Unity WebGL experience built from scratch as a custom interactive world.

🌍 Live Demo:
https://NanoRisk6.github.io/gwerGC/

Built with Unity WebGL and deployed via GitHub Pages.

This project is the foundation of an original universe focused on:

- Smooth performance
- Interactive systems
- Autonomous character concepts
- Experimental gameplay mechanics

Future Direction:
Expanding into larger worlds, multiplayer concepts, and eventually robotics-inspired autonomous systems.

---

Run it directly in your browser — no install required.
Best experienced on desktop.
